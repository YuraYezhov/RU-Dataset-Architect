# LostHeritage (Large)

## Цель
Автоматическая классификация типов утраченных культурных ценностей на основе их физических характеристик и текстовых описаний. Датасет предназначен для отработки навыков очистки «грязных» данных (data cleaning), проектирования признаков из текста (NLP-подход) и решения задач многоклассовой классификации.

## Предметная область
Сведения из официального реестра пропавших, утраченных, похищенных культурных ценностей Российской Федерации. Набор данных включает объекты живописи, иконописи, нумизматики и архивные документы.

## Файлы
* `data/lost_heritage_large_dataset_easy.csv` — итоговый датасет с очищенными признаками и целевой переменной (версия Easy).
* `data/lost_heritage_large_dataset_raw.csv` — «сырая» версия данных с пропусками, единицами измерения в строках и избыточным текстом.
* `meta/lost_heritage_large.meta` — технический паспорт датасета.
* `samples/01_load_dataset.pas` — пример загрузки и вывода данных в PascalABC.NET.
* `samples/02_describe_dataset.pas` — пример вывода схемы данных и первичного анализа.
* `samples/03_DecisionTreeClassifier_model.pas` — пример построения модели классификации Decision Tree.
* `samples/03_RandomForestClassifier_model.pas` — пример построения модели классификации Random Forest. 
* `eda_analysis.ipynb` — файл с полным аналитическим циклом: исследованием всех версий датасета (исходный, raw, easy), визуализацией процесса предобработки (преобразование из raw в easy) и обучением моделей машинного обучения.

## Задачи машинного обучения
* **Классификация**: предсказание категории культурной ценности (`target_label`).

## Признаки
| Имя | Тип | Описание | Роль |
| :--- | :--- | :--- | :--- |
| `height` | float | Высота объекта, приведенная к сантиметрам | Признак |
| `width` | float | Ширина объекта, приведенная к сантиметрам | Признак |
| `length` | float | Длина объекта, приведенная к сантиметрам | Признак |
| `weight` | float | Вес объекта, приведенный к граммам | Признак |
| `is_painting_hint` | int | Бинарный признак наличия ключевых слов живописи | Признак |
| `is_icon_hint` | int | Бинарный признак наличия ключевых слов иконописи | Признак |
| `is_coin_hint` | int | Бинарный признак наличия ключевых слов нумизматики | Признак |
| `is_doc_hint` | int | Бинарный признак наличия ключевых слов документов | Признак |
| `target_label` | int | Код категории (0: ART, 1: DOCS, 2: ICON, 3: COIN) | Целевая переменная |

## Целевая переменная
`target_label` — целое число от 0 до 3, представляющее собой закодированную группу классификации культурной ценности.

## Особенности данных
Исходные данные характеризуются высокой степенью разреженности (пропуски в габаритах до 90%). Реализована очистка текстовых артефактов и восстановление числовых значений из описаний. Используются временные признаки (DateTime) регистрации объектов.

## Использование
Данные могут быть загружены в среду PascalABC.NET.

## Ограничения
Объем выборки составляет 7550 объект. Ввиду специфики исходного реестра, точность классификации существенно опирается на качество текстовых описаний, тогда как геометрические параметры часто восстанавливаются по медианным значениям групп.
