# Data sources

Для формирования датасета **LostHeritage (Large)** были использованы следующие открытые данные:

* **Название источника:** Реестр пропавших, утраченных, похищенных культурных ценностей (Министерство культуры Российской Федерации).
* **Ссылка:** [https://opendata.mkrf.ru/opendata/7705851331-heritage_lost_objects](https://opendata.mkrf.ru/opendata/7705851331-heritage_lost_objects)
* **Дата обращения:** 11.07.2026.
* **Лицензия:** Открытые данные РФ (бесплатное использование в любых целях).

### Выполненные преобразования:

* **Отбор данных:** Из общего реестра объемом около 23 086 записей с помощью системы фильтрации по ключевым категориям было отобрано 7 550 объект, относящихся к четырем целевым группам: Живопись (ART), Иконы (ICON), Нумизматика (NUMISMATICS) и Архивные документы (DOCUMENTS).
* **Извлечение признаков:** На базе текстового анализа названий и описаний предметов реализована система выявления характерных признаков. С помощью ИИ-агента сформирован оптимизированный словарь основ слов (`keywords_dict_v2.py`) для расчета 4 бинарных признаков:
    *   маркеры живописи (is_painting_hint);
    *   маркеры иконописи (is_icon_hint);
    *   маркеры нумизматики (is_coin_hint);
    *   маркеры архивных документов (is_doc_hint).
* **Парсинг габаритов:** С помощью регулярных выражений (Regex) проведено извлечение числовых значений из неструктурированного текста описаний. Реализована конвертация исторических (дюймы, вершки) и метрических (мм, м) единиц измерения в единый стандарт (сантиметры и граммы).
* **Очистка и импутация:** Проведена очистка данных от информационного шума и артефактов парсинга. Для обеспечения целостности данных в версии **Easy** оставшиеся пропуски в колонках размеров заполнены глобальными медианными значениями.
* **Подготовка разметки:** На базе исходных классификаций реестра сформирована целевая переменная (`target_label`), закодированная методом Label Encoding (0 — ART, 1 — DOCUMENTS, 2 — ICON, 3 — NUMISMATICS).
* **Финальная сборка:** Сформирована итоговая таблица в формате CSV, объединяющая геометрические метрики, текстовые хинты и категориальную разметку для задач классификации.
