# HandwritingMetrics (Minimal)

## Цель
Автоматическая оценка длины рукописного слова на основе его геометрических характеристик. Датасет предназначен для отработки алгоритмов линейной регрессии и методов проектирования признаков (feature engineering) в задачах компьютерного зрения.

## Предметная область
Изображения русских рукописных слов, выполненных в едином каллиграфическом стиле на чистом белом фоне без шумов и артефактов.

## Файлы
* `data/handwriting_minimal_dataset.csv` — итоговый датасет с рассчитанными признаками и целевой переменной (100 объектов).
* `data/labels.csv` — файл разметки для отобранных 100 объектов (соответствие имени файла и написанного слова).
* `data/original_full.csv` — оригинальный файл аннотаций из исходного набора данных (4000 объектов).
* `meta/handwriting_minimal.meta` — технический паспорт датасета.
* `samples/01_load_dataset.pas` — пример загрузки и вывода данных в PascalABC.NET.
* `samples/02_describe_dataset.pas` — пример вывода схемы данных и первичного анализа.
* `samples/03_baseline_model.pas` — пример построения простейшей модели линейной регрессии.

## Задачи машинного обучения
* **Регрессия**: предсказание количества букв (`target_letters`).

## Признаки
| Имя | Тип | Описание | Роль |
| :--- | :--- | :--- | :--- |
| `file_name` | string | Идентификатор файла изображения | Технический |
| `aspect_ratio` | float | Соотношение сторон описывающего прямоугольника слова | Признак |
| `ink_density` | float | Относительная плотность заполнения области чернилами | Признак |
| `components` | int | Количество раздельных графических элементов | Признак |
| `peaks` | int | Количество значимых пиков вертикальной проекции | Признак |
| `solidity` | float | Степень заполнения выпуклой оболочки слова | Признак |
| `target_letters` | int | Фактическое количество символов в слове | Целевая переменная

## Целевая переменная
`target_letters` — целое положительное число, представляющее собой точное количество букв в слове.

## Особенности данных
Пропуски и выбросы отсутствуют. Данные высококачественные и однородные. Временные признаки (DateTime) и категориальные признаки не используются.

## Использование
Данные могут быть загружены в среду PascalABC.NET

## Ограничения
Объем выборки ограничен 100 объектами. Данные «стерильны» (отсутствует фоновый шум, наклон или разный цвет чернил), что упрощает задачу для учебных целей, но не отражает сложность реальных «грязных» рукописей.
