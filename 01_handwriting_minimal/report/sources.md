# Data sources

Для формирования датасета **HandwritingMetrics (Minimal)** были использованы следующие открытые данные:

* **Название источника:** Russian Cursive Handwriting (набор данных для распознавания русского рукописного текста).
* **Ссылка:** [https://www.kaggle.com/datasets/constantinwerner/russian-cursive-handwriting](https://www.kaggle.com/datasets/constantinwerner/russian-cursive-handwriting)
* **Дата обращения:** 05.07.2026.
* **Лицензия:** CC0: Public Domain.

### Выполненные преобразования:

* **Отбор данных:** Из исходного набора изображений (папка `Train`) было отобрано 100 образцов рукописных слов.
* **Извлечение признаков:**  С помощью библиотеки OpenCV реализован класс для анализа изображений. Внутри каждого метода извлечения признаков производится локальная бинаризация (порог 127, инверсия) для отделения чернил от фона. Рассчитаны 5 признаков: 
    *   соотношение сторон (aspect_ratio);
    *   плотность чернил (ink_density);
    *   количество связных компонентов (components);
    *   количество вертикальных пиков (peaks);
    *   плотность формы (solidity).
* **Подготовка разметки:** На базе метаданных источника был сформирован файл аннотаций (100 строк). Целевая переменная (`target_letters`) рассчитана как количество символов в строке названия для каждого файла.
* **Финальная сборка:** Сформирована итоговая таблица в формате CSV, объединяющая рассчитанные метрики и фактическую длину слов.
