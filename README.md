# RU-Dataset-Architect

Данный репозиторий представляет собой комплексный проект по разработке датасетов. Проект демонстрирует полный цикл работы с данными: от извлечения математических признаков из изображений с помощью компьютерного зрения (OpenCV) до подготовки финальной документации и примеров использования в средах программирования Python и PascalABC.NET.

## Архитектура проекта

Репозиторий разделен на три независимых модуля, соответствующих требованиям к
типам датасетов:

1.  01_handwriting_minimal — Минимальный датасет (100 объектов). Геометрический анализ рукописного текста.
2.  02_domes_medium — Средний датасет (500-1000 объектов). Классификация архитектурных форм (куполов) Золотого кольца.
3.  03_heritage_large — Большой датасет (7000+ объектов). Анализ реестра утраченных культурных ценностей РФ.

## 01_handwriting_minimal: Метрики рукописного текста

### Описание
Датасет содержит 100 образцов русских рукописных слов, извлеченных из набора данных Russian Cursive Handwriting. С помощью кастомных алгоритмов OpenCV каждое изображение преобразовано в вектор числовых признаков.

### Технические детали
  - Задача: Регрессия (прогнозирование количества букв в слове).
  - Метод извлечения: Локальная бинаризация и геометрический анализ контуров.
  - Ключевые признаки:
      - aspect_ratio: Пропорции описывающего прямоугольника.
      - ink_density: Плотность заполнения области чернилами.
      - peaks: Количество вертикальных штрихов (основной предиктор длины).
      - components: Количество связных фрагментов чернил.
      - solidity: Степень заполнения выпуклой оболочки.

## 02_domes_medium: Архитектура Золотого кольца

### Описание
Набор данных, посвященный классификации типов куполов православных храмов (луковичные, шлемовидные, шатровые). Данные собраны путем автоматизированного сбора изображений с портала sobory.ru.

### Технические детали
  - Задача: Многоклассовая классификация.
  - Метод извлечения: Анализ кривизны контуров и соотношения ширины купола к
    ширине барабана.
  - Версионность:
      - Raw: Содержит необработанные метрики, включая шумы и ошибки детекции.
      - Easy: Оптимизированный набор признаков, прошедший процедуру
        нормализации.

## 03_heritage_large: Реестр утраченных ценностей

### Описание
Масштабный датасет на основе государственного реестра похищенных и утраченных культурных ценностей РФ. Включает в себя более 23 000 записей.

### Технические детали
  - Задача: Классификация типа объекта (живопись, архивные документы, нумизматика).
  - Метод извлечения: Текстовый парсинг описаний, извлечение физических размеров и ключевых слов (материалы, техники).
  - Особенности: Демонстрирует работу с «грязными» текстовыми данными, обработку пропусков и приведение физических величин к единому формату.

## Технологический стек
Разработка и извлечение данных:
  - OpenCV: Предобработка изображений, поиск контуров, бинаризация и извлечение дескрипторов.
  - Pandas: Формирование структуры датасетов, агрегация данных и экспорт в CSV.
  - NumPy: Математические операции с матрицами изображений.
  - Scikit-learn: Построение базовых моделей для верификации качества признаков.

## Примеры использования (PascalABC.NET)
  - MLABC: Использование специализированной библиотеки для загрузки данных, проведения дескриптивного анализа и обучения моделей машинного обучения в учебных целях.

## Структура папок в каждом датасете
Каждый из трех модулей организован в соответствии с требованиями к отчетным
материалам:
  - /data: CSV-файлы (сырые и обработанные).
  - /meta: Технический паспорт датасета в формате .meta.
  - /scripts: Исходный код на Python для генерации данных.
  - /samples: Примеры программ на PascalABC.NET.
  - /report: Сопроводительная документация (README, Quality Report, Sources, AI Log).
